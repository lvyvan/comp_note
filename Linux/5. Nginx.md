## 1.3 Nginx 基础配置

```conf
# 开启一个 work 进程，通常与 CPU core 相同
worker_processes  1;

events {
    worker_connections  1024;
}

http {
	# 请求头类型
    include       mime.types;
    # 如果没有包含在请求头，则默认以application/octet-stream;解析
    default_type  application/octet-stream;

	# 复制0拷贝 系统调用
    sendfile        on;
	
	# 保持链接
    keepalive_timeout  65;

	# 主机 vhost
    server {
	    # 监听端口号
        listen       80;
        # 域名 | 主机名
        server_name  localhost;

		# URI
        location / {
	        # 相对于 主目录(/usr/local/nginx/)路径
            root   html;
            # 默认页
            index  index.html index.htm;
        }

		# 错误页
        error_page   500 502 503 504  /50x.html;
        location = /50x.html {
            root   html;
        }
    }
}
```

## 1.8 URL Rewrite

rewrite \<regex\> \<replacement\> \[flag\];

flag标记说明

- last 本规则匹配完成后，继续向下匹配新的 lcation URL 规则
- break 本规则匹配完成即终止，不再匹配后面的任何规则
- redirect 返回302临时重定向，浏览器会显示跳转后的URL地址
- permanent 返回301永久重定向，浏览器会显示跳转后的URL地址

```conf 
worker_processes  1;

events {
    worker_connections  1024;
}

http {
    include       mime.types;
    default_type  application/octet-stream;

    sendfile        on;
	
    keepalive_timeout  65;

	upstream httpd {
	}

    server {
        listen       80;
        server_name  localhost;

		# URI
        location / {
	        rewrite ^/([0-9]+).html$ /index.jsp?pageNum=$1 break;
	        proxy_pass http://192.168.1.102:80;
        }

        location ~*/(js|img|css ) {
			root html
			index index.html
        }

        error_page   500 502 503 504  /50x.html;
        location = /50x.html {
            root   html;
        }
    }
}
```
